Setup iCloud / Dropbox / Google Drive sync

Create a symlink to sync backups with cloud storage:

iCloud (macOS): mkdir -p ~/Library/Mobile\ Documents/com~apple~CloudDocs/firefox-frontpage-backup && ln -sf ~/Library/Mobile\ Documents/com~apple~CloudDocs/firefox-frontpage-backup ~/Downloads/firefox-frontpage-backup
Dropbox: mkdir -p ~/Dropbox/firefox-frontpage-backup && ln -sf ~/Dropbox/firefox-frontpage-backup ~/Downloads/firefox-frontpage-backup
Google Drive: mkdir -p ~/Google\ Drive/My\ Drive/firefox-frontpage-backup && ln -sf ~/Google\ Drive/My\ Drive/firefox-frontpage-backup ~/Downloads/firefox-frontpage-backup

Run the command once in Terminal, then enable Auto Backup above.
